# Ziny.io Headless Rebuild

Static frontend for ziny.io, built with Astro, content from WordPress REST API.

## Quick Start

npm install
npm run dev

## Project Structure
- `/src/pages/` — Astro pages (routes)
- `/src/components/` — Reusable UI components
- `/src/layouts/` — Page layouts
- `/src/lib/` — WordPress API client and utilities
- `/public/` — Static assets (images, fonts)
